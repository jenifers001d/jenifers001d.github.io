Python 自學第八天：Package 封包

發表於 2019-12-07 更新於 2019-12-09 分類於 Python
AdSense

功用

模組其實就是 .py 的檔案。而 Package 封包，是用來放置數個模組的資料夾，方便模組的管理和使用。

在 Package 中，形成自己的命名空間，可以採用 . 的方式，引用 Main Package 裡面的子封包、模組、變數和函式，就不用擔心名稱重複的問題。

使用封包後，就可以不需要自己手動增加模組搜尋路徑到 sys.path 中。超級方便！！！！！
