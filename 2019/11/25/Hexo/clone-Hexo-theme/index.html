發布 : 2019-11-25 分類 : Hexo 瀏覽 :

Hexo 有許多協作開發者，他們將自己的主題樣式放置在官網上，到目前為止，已經有 277 個主題可供使用者下載。

在你的 Hexo 資料夾中，可以看到有一個 themes 資料夾，裡面有預設的主題 landscape。此後下載的主題，都會放到 themes 資料夾中。

後記

網路上有很多使用 NextT 的相關文章，這邊大推薦 Ray 的 Hexo 系列。

我當初因為不想要和大多數人一樣使用 NextT，加上 NextT 是使用 Swig 撰寫 (我不會 >皿< )，因此拖延了半年，我的網站最近才上線。我當初是選中了 BMW 這個主題，但是開發者好像已經很久沒有更新了…但是我也懶得再選另一個主題了，心太累。
