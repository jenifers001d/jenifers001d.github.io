設定 Hexo 基本配置

Posted on 2019-11-25 Edited on 2019-11-26 In Hexo

配置文件常以 YAML 語法撰寫，YAML (Yet Another Markup Language) 是一個可讀性高，被設計用來表達資料序列化的格式。它跟 JSON 格式非常的相近。

有兩大資料格式：

  • key: value 對 (map)。注意：冒號為半形+空白字元+value。
  • 短槓+空白字元，表示陣列 (list)。

特點：

  • 使用縮進表示層級，同層級要對齊。
  • 不可以用 tab，只能用空白鍵。
