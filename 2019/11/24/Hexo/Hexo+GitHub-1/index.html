Posted on 2019-11-24 Edited on 2019-12-03 In Hexo Views: Disqus:

Hexo 是什麼？

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown(一種簡單語法的文本語言，能讓使用者輕鬆編寫，並轉換成 HTML 文件) 解析文章，可以讓使用者專注在文章撰寫。

短時間內，透過漂亮的主題(套用他人主題，或是自己做主題)產生靜態檔案(只有前端，也可以架設網誌和部落格！)。

Hexo 支援許多強大的外掛和擴充系統，例如：搜尋、留言板、Facebook 社群工具等。

以 Node.js 為核心，需要預先編譯，才能丟到 GitHub 上 (也就是遠端伺服器或網站上)，但是優點是一旦網頁內容很大，會比 Jekyll 速度快 (Ruby 為核心，不需要先編譯)。

台灣人製作的！！！官方網站都有支援繁體中文，快去看看吧 >> Hexo
