設定 GitHub 和 Hexo 配置

建立 GitHub 儲存庫

從此開始分成兩種類型，取決於你想要將網站設在哪一種網址中，依照最初儲存庫名稱輸入時，分為：

  • 用戶名
    • 網址：https://{用戶名稱}.github.io
    • 每個帳戶只有一個，能放在最根目錄的專案
    • 部署Branch：master
  • 專案名
    • 網址：https://{用戶名稱}.github.io/{專案名稱}
    • 任意、無數量限制
    • 部署Branch：gh-pages
