Python 自學第三天：數字、字串的基本運算

發布 : 2019-11-30 分類 : Python 瀏覽 :

數字

使用加 +、減 -、乘 * 和除 / 做基本運算。

  • 整數除法：

除法的餘數

使用 %。

指數 (開根號、平方、三次方等)

使用 **。

字串

截取字串

可以把一個字串想成是一組 Tuple，每個字母都有對應的位置 (index)，從 0 開始編號，如下圖。

用中括弧 [] 取得該位置的字元。用 [n:m] 截取從 n 到 m-1 的一段字串。

字串串接

使用 +。

重複輸出

使用 *。

多行輸出

使用 \n、'''可換行''' (三個單引號包圍) 或 """可換行""" (三個雙引號包圍)。

包含、不包含

使用 in 和 not in。

輸出原始字串，移除特殊意義

使用 r 或 R 放在字串第一個引號前面。

字串格式化

使用 % 開頭表示即將插入的變數。
