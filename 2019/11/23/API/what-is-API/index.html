API еҲ°еә•жҳҜд»Җйәјпјҹ

зҷјеёғ : 2019-11-23 еҲҶйЎһ : API зҖҸиҰҪ :

еҗҚи©һи§ЈйҮӢ

йҰ–е…ҲжҲ‘еҖ‘е…ҲйҮҗжё…е№ҫеҖӢеҗҚи©һ

  • Method/Functionпјҡ
    Java/C# зЁұзӮә MethodпјӣC/C++ зЁұзӮә FunctionпјҢдҪҶдёҚе®Ңе…ЁзӣёеҗҢгҖӮ
    (еңЁзү©д»¶е°Һеҗ‘дёӯзҡ„ method жңүеҲҶе…©зЁ®пјҡдёҖзЁ®жҳҜйңҖиҰҒе…ҲеҜҰй«”еҢ–дёҖеҖӢзү©д»¶гҖӮдёҖзЁ®жҳҜеҸҜд»ҘзӣҙжҺҘдҪҝз”ЁгҖӮеүҚиҖ…жҲ‘еҖ‘зЁұзӮә methodпјҢеҫҢиҖ…жҲ‘еҖ‘зЁұзӮә static methodгҖӮиҖҢ function е°ҚжҮүеҲ°зҡ„жҳҜ static methodгҖӮ)

  • Libraryпјҡ
    еҮҪејҸеә«пјҢиЎЁзӨәе°ҮдёҖе Ҷе·Із¶“еҜ«еҘҪзҡ„ method/function еҢ…жҲҗдёҖеҖӢ libraryпјҢз•¶жҲ‘еҖ‘еңЁжҮүз”ЁзЁӢејҸзҡ„жңҖдёҠж–№ import(еј•з”Ё) йҖҷеҖӢ library еҫҢе°ұиғҪдҪҝз”Ёе…¶еҢ…еҗ«зҡ„ functionгҖӮеғҸжҳҜжҲ‘еҖ‘иҰҒдҪҝз”Ёзўҳй…’жҲ–й…’зІҫжЈүзүҮ (function) жңғе…ҲжӢҝеҮәйҶ«и—Ҙз®ұ (йҶ«и—Ҙз®ұlibrary)гҖӮ
    (жҲ–зЁұзЁӢејҸеә«гҖҒC# еҸҲзЁұзӮәйЎһеҲҘеә«)

иҲүеҖӢдҫӢеӯҗдҫҶиӘӘжҳҺпјҡ

е°ҸжҳҺжҮүе…¬еҸёиҰҒжұӮпјҢеҜ«дәҶдёҖеҖӢ C# йҒҠжҲІеј•ж“Һ(е№«еҠ©йҒҠжҲІиЁӯиЁҲиҖ…е®№жҳ“е’Ңеҝ«йҖҹең°еҒҡеҮәйҒҠжҲІзЁӢејҸзҡ„жҮүз”ЁзЁӢејҸ)гҖӮ
е…¶дёӯжңүдёҖеҖӢ method е®ҡзҫ©еҰӮдёӢ

Math.Round()гҖҒMath.Pow()пјҢжҳҜз”ұ C# е…§е»ә library жҸҗдҫӣзҡ„е…©еҖӢ static methodгҖӮ

RoundDoubleNumber()пјҢжҳҜе°ҸжҳҺеҜ«зҡ„ methodпјҢдёҰдё”е°ҸжҳҺеңЁе®ҳж–№иӘӘжҳҺж–Үд»¶дёӯпјҢи©ізҙ°иӘӘжҳҺеӨ–йғЁзЁӢејҸеҰӮдҪ•дҪҝз”Ё RoundDoubleNumber()гҖӮ

е°ҸжҳҺзҡ„еҗҢдәӢе°Ҹе…үиҰҒиЁӯиЁҲдёҖж¬ҫйҒҠжҲІ AпјҢйңҖиҰҒз”ЁеҲ°е°ҸжҳҺеҜ«зҡ„йҒҠжҲІеј•ж“ҺпјҢе°Ҹе…үдёҚйңҖиҰҒзҹҘйҒ“ RoundDoubleNumber() е…§йғЁзҡ„е®ҡзҫ©пјҢеҸӘйңҖиҰҒйҖҸйҒҺе®ҳж–№иӘӘжҳҺж–Үд»¶дҫҶзҹҘйҒ“з”ЁйҖ”е’ҢдҪҝз”Ёж–№жі•гҖӮ

иҖҢеғҸ RoundDoubleNumber() йҖҷжЁЈжҸҗдҫӣзөҰеӨ–йғЁжҮүз”ЁзЁӢејҸдҪҝз”Ёзҡ„иҮӘе®ҡзҫ© method/functionпјҢжҲ‘еҖ‘е°ұзЁұзӮә APIгҖӮе®ҳж–№иӘӘжҳҺж–Үд»¶еҸҲзЁұ API иӘӘжҳҺж–Үд»¶гҖӮ

з”ұдёҠйқўзҡ„дҫӢеӯҗеҫ—зҹҘ

  • API(Application Programming Interface)пјҡ
    жҮүз”ЁзЁӢејҸд»ӢйқўпјҢжҢҮе·Із¶“еҜ«еҘҪзҡ„ method/function жҸҗдҫӣзөҰеӨ–йғЁзЁӢејҸеӯҳеҸ–гҖӮ

иҲүдҫӢ

1. Google Map

Google Map жң¬иә«жҳҜеҖӢжҮүз”ЁзЁӢејҸпјҢжңүиЁұеӨҡзҡ„еҠҹиғҪпјҢдҫӢеҰӮжҹҘи©ўең°еқҖжҲ–жҳҜйЎҜзӨәиЎӣжҳҹең°ең–зӯүгҖӮ

з•¶ Google жғіжҠҠйғЁеҲҶеҠҹиғҪжҸҗдҫӣзөҰеӨ–йғЁзҡ„з¶Ій ҒжҮүз”ЁзЁӢејҸй–ӢзҷјиҖ…гҖҒжүӢж©ҹжҮүз”ЁзЁӢејҸй–ӢзҷјиҖ…пјҢе®ғдёҚеҝ…жҠҠзЁӢејҸзўјй–ӢжәҗзөҰеӨ§е®¶пјҢиҖҢжҳҜе°ҮжғіжҸҗдҫӣзҡ„еҠҹиғҪ (method) еҢ…жҲҗдёҖд»Ҫ Google Map APIгҖӮдёҰж’°еҜ«е®ҳж–№иӘӘжҳҺж–Үд»¶пјҢиӘӘжҳҺеҰӮдҪ•дҪҝз”ЁгҖӮ
